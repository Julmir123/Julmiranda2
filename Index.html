Semana 1


Esta semana fue una introducción al curso donde pude notar lo extenso del silabo desde los cursos de programación que abarcan el 1er y 2do promedio hasta el proyecto en Arduino del 3er promedio.

Sinceramente este curso me emociona mucho ya que la programación web es algo que me llamo la atención desde hace un tiempo y este curso fue el empujón necesario para poder explorar este mundo.

Además de lograr ser el delegado, logre armar un grupo que a mi parecer es muy capaz y ya tenemos algo muy claro de lo que será nuestro proyecto del 3er promedio.


Semana 2


Empezamos esta emocionante semana descargando el Visual Studio Code de Miicrosoft. Luego de inspeccionar algunos elementos de paginas web me di cuenta que la programación es un extenso y basto oceano del cual apenas he visto una playa.



Aprendimos ciertas etiquetas muy interesantes y trabajamos con una plantilla llena de fotos de cachorros. La plantilla me sirvió para ver la etiquetas en acción y entender que el html es en realidad la estructura de la pagina web.



Al ser alguien relativamente nuevo creo que fui capaz de entender lo básico del html. gracias a la intro hacia css pude investigar un poco sobre css de manera independiente, además de un de JS y de esa manera lograr la primera tarea del curso que justamente es esta bitácora.
